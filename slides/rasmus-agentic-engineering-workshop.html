Best Practices from the Trenches

✅ DO

  • Start in chat mode: "How should I build this?"
  • Use validation gates: Let it test and fix
  • Reference docs: /ai-docs folder
  • Small iterations: Features, not entire apps
  • Modern tools: UV for Python, not pip

❌ DON'T

  • Skip planning phase: Jump straight to code
  • Manual reviews: During AI generation
  • Legacy approaches: Old patterns/tools
  • Big bang builds: Entire applications
  • Ignore feedback loops: Test late or never
